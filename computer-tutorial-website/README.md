# TechLearn Pro - Professional Computer Tutorial Website

A comprehensive, professional computer tutorial website with interactive 3D visualizations, email automation, user dashboard, and advanced search functionality.

## ✨ Features

### Core Features
✅ **Professional & Corporate Design** - Blue color scheme, clean layout  
✅ **7 Tutorial Categories** - Programming, Web Dev, Hardware, OS, Networking, Cybersecurity, Basics  
✅ **3D Interactive Visualizations** - Computer architecture and data flow  
✅ **Advanced Search** - Find tutorials by topic, category, or description  
✅ **User Dashboard** - Track progress, enrolled courses, hours learned  
✅ **Email Newsletter** - Automated subscription system with localStorage  
✅ **Contact Forms** - With automated email responses  
✅ **Course Enrollment** - Track enrolled and completed courses  
✅ **Certificate Generation** - Download certificates upon completion  
✅ **Responsive Design** - Mobile-friendly, works on all devices  

### Technical Features
- Vanilla HTML, CSS, JavaScript (No frameworks required)
- Three.js integration ready
- Local storage for data persistence
- Professional animations and transitions
- SEO-friendly structure
- Accessible components

## 🛠️ How to Use

### 1. Access the Website
- Download all files to the same folder
- Open `index.html` in a modern web browser
- No server required - runs completely locally

### 2. Navigate Sections
- **Home** - Hero section with introduction
- **Tutorials** - Browse all 9 courses
- **Dashboard** - Track your progress
- **About** - Learn about TechLearn Pro
- **Contact** - Get in touch

### 3. Enroll in Courses
- Click any tutorial card to enroll
- Data stored in browser's localStorage
- Enrollment appears in dashboard

### 4. Subscribe to Newsletter
- Enter email in newsletter section
- Subscription stored locally
- Get notifications for new content

### 5. Complete Contact Form
- Fill in contact information
- Submit your message
- Data saved for follow-up

## 💾 Local Storage

The website uses browser localStorage to store:
- **Enrolled Courses**: `enrolledCourses` (array)
- **Completed Courses**: `completedCourses` (array)
- **Subscribers**: `subscribers` (email list)
- **Contact Data**: `contactData` (form submissions)

## 🔍 Search Features

- Real-time search as you type
- Search across:
  - Tutorial titles
  - Categories
  - Descriptions
- Click results to enroll
- Case-insensitive matching

## 📊 Dashboard Metrics

- **Courses Enrolled**: Number of enrolled courses
- **Completed**: Number of finished courses
- **Hours Learned**: Calculated from completed courses
- **Achievements**: Badges earned (1 per 2 courses)
